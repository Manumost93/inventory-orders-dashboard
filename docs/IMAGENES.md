# 📷 Instrucciones para añadir imágenes al README

Para completar el README, necesitas guardar las siguientes imágenes en la carpeta `docs/images/`:

## Imágenes necesarias:

1. **`dashboard-overview.png`** - Captura del dashboard principal con:
   - Métricas de ingresos ($459.40)
   - Total de órdenes (3)
   - Productos activos (2) 
   - Stock bajo (5)
   - Gráfico "Orders by Status"
   - "Top Customer" (Initech)
   - "Last 5 Orders"
   - "Inventory Snapshot"

2. **`products-management.png`** - Vista de la página de productos con:
   - Tabla de productos (Classic Burger, Cheese Burger, Vegan Burger)
   - Botones de búsqueda y filtros
   - Botón "ADD PRODUCT"
   - Columnas: Name, SKU, Price, Stock, Status, Created At, Actions

3. **`add-product-modal.png`** - Modal de añadir producto con:
   - Título "Add Product"
   - Campos: Product Name, Price
   - Botones: CANCEL, CREATE

## Cómo añadir las imágenes:

1. Guarda las capturas de pantalla que enviaste con los nombres exactos arriba
2. Colócalas en la carpeta: `docs/images/`
3. ¡Las referencias en el README ya están configuradas!

## Estructura final:
```
docs/
└── images/
    ├── dashboard-overview.png
    ├── products-management.png
    └── add-product-modal.png
```
